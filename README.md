# Nest GatewayQL

## Description

[Nest GatewayQL](https://github.com/GatewayQL/nest-GatewayQL) A microservices GraphQL gateway built on Nestjs.

## Installation

```bash
$ npm install
```

## Copy config, env and build project
```bash
$ cp .example.env .env

$ npm run build
```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
